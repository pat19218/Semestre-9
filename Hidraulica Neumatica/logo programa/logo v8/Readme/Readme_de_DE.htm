Allgemeines

Das Anlaufverfahren für Setup-Programm und LOGO!Soft Comfort kann auf älteren Computern eine Weile dauern, ohne dass auf dem Bildschirm eine Anzeige erscheint. Starten Sie die Anwendung nicht mehrmals.

Installation

Weil Java plattformspezifische Zeichen nicht immer korrekt verarbeitet, vermeiden Sie bitte die Verwendung von Sonderzeichen wie ! / : ; \ . ? ' " in Verzeichnisnamen.

Sie können LOGO!Soft Comfort auch installieren, indem Sie das Verzeichnis, wie in Schritt 1 unten beschrieben, auf Ihre Festplatte kopieren. Der Name des ausführbaren Programms ist LOGOComfort.exe unter MS-Windows, LOGOComfort.app unter Mac-OSX und LOGOComfort unter Linux.

Wenn Sie LOGO!Soft Comfort auf einen Server kopieren möchten, gehen Sie folgendermaßen vor:

  1. Kopieren Sie den Inhalt des folgenden Verzeichnisses von der LOGO!Soft DVD-ROM auf den Server.

l        <DVD-ROM>: \Windows\Application_32 oder \Windows\Application_64

l        <DVD-ROM>: \Mac\Application

l        <DVD-ROM>: \Linux\Application_32 oder \Linux\Application_64 

  1. Richten Sie auf Client-Seite eine Verbindung zu diesem Serververzeichnis ein.

Wenn Sie LOGO!Soft Comfort über das Setup-Programm auf dem Server installiert haben, müssen Sie alle absoluten Pfadangaben in der Datei *.lax (z.B. lib\classes.jar) in Pfade in Bezug auf das Installationsverzeichnis ändern.

Sie können die Datei auch aus dem Verzeichnis <Betriebssystem\Anwendung>, z.B. \Windows\Application_32, von der LOGO!Soft Comfort DVD-ROM kopieren.

Deinstallation

Sie können LOGO!Soft Comfort auf allen Plattformen mit dem Deinstallationsprogramm deinstallieren. Auf MS-Windows-Plattformen beispielsweise befindet sich dieses Programm unter

<Installationsverzeichnis>\UninstallerData\Uninstall.exe.

Weitere Sprachen

Für LOGO!Soft Comfort werden sechs Sprachen installiert. Die Standardsprache wählen Sie während des Installationsvorgangs aus. Sprachen können Sie wie folgt ändern:

  1. "Extras" à "Optionen" in der Menüleiste.
  2. Gewünschte Sprache im Optionsfenster wählen, auf "OK" klicken.
  3. LOGO!Soft Comfort neu starten.

Bekannte Probleme

1.      Kommunikationfehler können auftreten, wenn Sie LOGO! Hardware mit LOGO!Soft Comfort unter MS-Windows Plug&Play-Betriebssystemen betreiben. Zur Behebung dieses Problems gehen Sie folgendermaßen vor, um die Länge des FIFO-Puffers für den COM-Port weitestgehend zu verringern:

1)      Klicken Sie auf "Eigenschaften" und öffnen Sie das Register "Porteinstellungen".

2)      Klicken Sie auf "Erweitert" und setzen Sie dann Empfangspuffer und Sendepuffer auf "Niedrig".

2.      Wenn die Linien der Seitenränder nicht gedruckt werden, ändern Sie die Seitenränder über "Datei > Seite einrichten".

3.      Wenn Sie einen Drucker Kyocera FS1600 verwenden, wird der in Zeichnungen enthaltene Text möglicherweise fehlerhaft gedruckt. Um dieses Problem zu beheben, installieren Sie den Druckertreiber "Kyocera FS1600+ (KPDL2)". Eine englische Version dieses Treibers finden Sie auf Ihrer DVD unter <DVD-ROM>:/windows/printer/k9xpsen.exe.

Einzelheiten zum Installieren von Druckertreibern finden Sie in der Dokumentation Ihres Betriebssystems.

4.      Zur Behebung von Druckerproblemen aktualisieren Sie Ihren Druckertreiber oder installieren den Standardtreiber von der CD Ihres Betriebssystems.

5.      Sind die Druckprobleme dadurch nicht behoben, können Sie die Datei im PDF-Format speichern und im Adobe Acrobat Reader ausdrucken. Sie können Ihre Zeichnungen auch im BMP- oder JPG-Format speichern oder in anderen Anwendungen bearbeiten.

6.      Um das Flackern des Cursors im Simulationsmodus zu beheben, deaktivieren Sie benutzerspezifische Mauseinstellungen bzw. die Mausspur.

7.      Wenn der Name des Installationsverzeichnisses der untersten Ebene auf einem Windows-Betriebssystem ein Leerzeichen enthält, kann das System LOGO!Soft Comfort nicht öffnen. Vergewissern Sie sich, dass Sie keine Leerzeichen im Namen des Installationsverzeichnisses der untersten Ebene verwenden.

Macintosh

Voraussetzungen

Sie haben Uploads/Downloads von Steuerungsprogrammen von/zu der LOGO! Hardware mit dem Keyspan USB-PDA-Adapter (Keyspan USA-19) und dem LOGO! USB-PC-Kabel getestet.

Wenn LOGO!Soft Comfort nicht mit Ihrem LOGO! Basismodul über die serielle Schnittstelle, die dem Keyspan USA-19 oder dem LOGO! USB-PC-Kabel entspricht, kommunizieren kann, prüfen Sie, ob in Ihrem Dateisystem der Ordner /var/lock vorhanden ist. Ist er nicht vorhanden, legen Sie diesen Ordner an und stellen sicher, dass Sie vollständige Zugriffsrechte dafür haben. Wenn Sie zuvor eine Vorgängerversion des Pakets RXRXComm genutzt haben, müssen Sie möglicherweise übrig gebliebene Verriegelungsdateien im Vorgängerordner /var/spool/uucp löschen. Prüfen Sie diese Ordner, bevor Sie die Setup-Anwendung durchführen.

Installation

Wenn Sie das Setup-Programm Setup_Mac.command nicht ausführen können, haben Sie möglicherweise nicht die Berechtigung dafür. Dies kann beispielsweise der Fall sein, wenn Sie Setup_Mac.command von einem Windows-Computer kopieren. 

Um die Benutzerattribute für die Datei Setup_Mac zu ändern, gehen Sie folgendermaßen vor:

Wenn Sie Setup_Mac.command starten, erhalten Sie eine Fehlermeldung wie "Befehlsdatei öffnen...". Quittieren Sie diese Fehlermeldung. Ändern Sie im nächsten Terminalfenster zunächst die Benutzerattribute durch Eingabe des folgenden Befehls an der Eingabeaufforderung:

chmod a+x Setup_Mac.command

Geben Sie dann die Zeichenfolge ./Setup_Mac.command ein, um das Setup-Programm zu starten. Schließen Sie außerdem das Terminalfenster und doppelklicken Sie auf die ausführbare Datei, um das Setup-Programm aufzurufen. Beachten Sie, dass die Groß- und Kleinschreibung im Dateinamen relevant ist.

Drucken

MAC OS-X unterstützt das Drucken im Querformat nicht. Wenn Sie einen Ausdruck in diesem Format benötigen, speichern Sie die Zeichnung über den Befehl "Speichern unter" im PDF-Format und drucken Sie sie dann.
