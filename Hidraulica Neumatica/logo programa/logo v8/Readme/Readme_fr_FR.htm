Généralités

La procédure de démarrage pour le programme Setup et LOGO!Soft Comfort peut durer assez longtemps sur des ordinateurs anciens sans qu’un message ne s'affiche à l'écran. Ne démarrez pas l’application plusieurs fois.

Installation

Java ne traitant des toujours des caractères spécifiques à une plate-forme correctement, veuillez éviter d’utiliser des caractères spéciaux tels que ! / : ; \ . ? ' " dans les noms de répertoire.

Vous pouvez également installer LOGO!Soft Comfort en copiant le répertoire sur votre disque dur comme décrit ci-dessous à l’étape 1. Le nom du programme exécutable est LOGOComfort.exe sous MS-Windows, LOGOComfort.app sous Mac-OSX et LOGOComfort sous Linux.

Procédez comme suit si vous voulez copier LOGO!Soft Comfort sur un serveur :

  1. Copiez le contenu du répertoire suivant du DVD-ROM LOGO!Soft sur le serveur.

l        <DVD-ROM>: \Windows\Application_32 ou \Windows\Application_64

l        <DVD-ROM>: \Mac\Application

l        <DVD-ROM>: \Linux\Application_32  ou \Windows|Application_64 

  1. Etablissez côté client une liaison vers ce répertoire du serveur.

Si vous avez installé LOGO!Soft Comfort sur le serveur via la procédure setup, vous devez changer toutes les entrées de chemin absolues dans le fichier .lax (par exemple, lib\classes.jar) en des chemins indiquant le répertoire d'installation.

Vous pouvez également copier le fichier à partir du dossier <système d'exploitation\Application>, par exemple Windows\Application_32, du DVD-ROM LOGO!Soft Comfort.

Désinstallation

Vous pouvez désinstaller LOGO!Soft Comfort de toutes les plates-formes à l’aide du programme de désinstallation. Ce programme se trouve par ex. pour toutes les plates-formes MS-Windows sous

<Répertoire d’installation>\UninstallerData\Uninstall.exe.

Autres langues

Par défaut, six langues sont installées pour LOGO!Soft Comfort. Et la langue par défaut est celle que vous avez choisie lors de l'installation. Vous pouvez changer les langues ainsi:

1.      Cliquez "Outils"à"Options" ds barre menu.

2.      Choisissez la langue voulue ds la fenêtre et cliquez sur "OK".

3.      Relancez LOGO!Soft Comfort.

Problèmes connus

1.      Des erreurs de communication peuvent se produire si vous utilisez du matériel LOGO! avec LOGO!Soft Comfort sous des systèmes d’exploitation MS-Windows Plug&Play. Pour corriger ce problème, procédez comme suit pour réduire autant que possible la taille du tampon FIFO pour le port COM :

1)      Cliquez sur „Propriétés“ et ouvrez l’onglet "Paramètre du port".

2)      Cliquez sur "Avancé" et réglez le tampon de réception et le tampon d’émission sur "Faible".

2.      Si les lignes des marges ne sont pas imprimées, modifiez les marges via la commande "Fichier > Mise en page".

3.      Si vous utilisez une imprimante Kyocera FS1600, il se peut que les textes contenus dans des graphiques ne soient pas imprimés correctement. Pour corriger ce problème, installez le pilote d’imprimante "Kyocera FS1600+ (KPDL2)". Vous trouverez une version anglaise de ce pilote sur votre DVD sous <DVD-ROM>:/windows/printer/k9xpsen.exe.

Vous trouverez des détails sur l’installation de pilotes d’imprimante dans la documentation de votre système d’exploitation.

4.      Pour corriger des problèmes d’imprimante, actualisez votre pilote d’imprimante ou installez le pilote standard à partir du CD de votre système d'exploitation.

5.      Si cela ne suffit pas à régler les problèmes d’imprimante, vous pouvez sauvegarder le fichier au format PDF et l’imprimer dans l’Adobe Acrobat Reader. Vous pouvez également sauvegarder vos graphiques au format BMP ou JPG ou travailler dans d’autres applications.

6.      Pour arrêter le clignotement du curseur en mode de simulation, désactivez les paramètres utilisateur de la souris ou la trace de souris.

  1. Si le nom d’un répertoire d’installation racine sur un système d’exploitation Windows contient un espace, le système ne peut pas ouvrir LOGO!Soft Comfort. Veillez à ne pas utiliser de caractères d’espacement dans le nom du répertoire d’installation racine.

Macintosh

Conditions

Vous avez testé le chargement/téléchargement de programmes de commande dans le/du matériel LOGO! avec l’adaptateur USB-PDA Keyspan (Keyspan USA-19) et le câble PC USB LOGO!.

Si LOGO!Soft Comfort ne peut pas communiquer avec votre module de base LOGO! via l’interface série correspondant au Keyspan USA-19 ou au câble PC USB LOGO!, vérifiez si le répertoire /var/lock existe dans votre système de fichiers. S'il n'existe pas, créez ce répertoire et assurez-vous que vous avez des droits d’accès complets. Si vous avez auparavant utilisé une version antérieure du paquet RXRXComm, vous devez éventuellement supprimer des fichiers de verrouillage dans le répertoire de l’ancienne version /var/spool/uucp. Contrôlez ces répertoires avant l’exécution du Setup.

Installation

Si vous ne pouvez pas exécuter le programme setup Setup_Mac.command, il se peut que vous ne possédiez pas les droits nécessaires. Ceci peut être le cas par ex. lorsque vous copiez le programme Setup_Mac.command à partir d’un ordinateur Windows. 

Procédez comme suit pour modifier les attributs utilisateur pour le fichier Setup_Mac :

Lorsque vous démarrez Setup_Mac.command, un message d’erreur s’affiche, un message d’erreur, comme par ex. "Ouvrir le fichier de commandes...". Acquittez ce message d’erreur. Modifiez dans la fenêtre de terminal suivante d’abord les attributs utilisateur en saisissant la commande suivante dans l’invite de commandes :

chmod a+x Setup_Mac.command

Saisissez ensuite le jeu de caractères ./Setup_Mac.command pour démarrer le programme Setup. Fermez également la fenêtre de terminal et double-cliquez sur le fichier exécutable afin d’appeler le programme Setup. Tenez compte du fait que la casse doit être respectée dans les noms de fichiers.

Imprimer

MAC OS-X prend en charge l’impression au format Paysage. Si vous avez besoin d’une impression dans ce format, sauvegardez le graphique via la commande "Enregistrer sous" au format PDF et imprimez-le ensuite.
