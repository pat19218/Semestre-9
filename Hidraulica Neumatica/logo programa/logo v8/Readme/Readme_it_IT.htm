Informazioni generali

Nei computer meno recenti, dopo che è stata lanciata la procedura di avvio del programma di setup e di LOGO!Soft Comfort può passare del tempo prima che compaia qualcosa sullo schermo. Non avviare l'applicazione più volte.

Installazione

Poiché i caratteri specifici della piattaforma Java non vengono sempre interpretati correttamente, è consigliabile evitare l'uso di caratteri speciali quali ! / : ; \ . ? ' " nei nomi delle directory.

LOGO!Soft Comfort può essere installato anche copiando la directory nel disco fisso come descritto più avanti al punto 1. Il nome del programma eseguibile è LOGOComfort.exe in MS Windows, LOGOComfort.app in Mac OSX e LOGOComfort in Linux.

Per copiare LOGO!Soft Comfort su un server procedere nel seguente modo:

  1. Copiare il contenuto delle seguenti directory dal DVD-ROM di LOGO!Soft nel server.

l        <DVD-ROM>: \Windows\Application_32 oppure \Windows\Application_64

l        <DVD-ROM>: \Mac\Application

l        <DVD-ROM>: \Linux\Application_32 oppure \Linux\Application_64  

  1. Dal client stabilire un collegamento con la directory scelta.

Se si è installato LOGO!Soft Comfort sul server tramite setup, è necessario modificare tutte le voci dei percorsi assoluti contenute nel file .lax (ad es. lib\classes.jar) in percorsi relativi alla directory di installazione.

In alternativa è possibile copiare il file da <Sistema operativo\Applicazione>, ad es. \Windows\Application_32, del DVD-ROM di LOGO!Soft Comfort.

Disinstallazione

LOGO!Soft Comfort può essere disinstallato da qualsiasi piattaforma mediante il programma di disinstallazione. Sulle piattaforme Windows il programma si trova ad esempio in

<directory di installazione>\UninstallerData\Uninstall.exe.

Altre lingue

Vengono installate per default sei lingue per LOGO!Soft Comfort. La lingua selezionata durante l'installazione è quella di default. Per cambiare lingua:

1.      Selez. "Strumenti"à"Opzioni" (barra menu).

2.      Selezionare la lingua e "OK" nella finestra delle opzioni.

3.      Riavviare LOGO!Soft Comfort.

Problemi noti

1.      Se si utilizza l'hardware LOGO! con LOGO!Soft Comfort in un sistema plug & play di MS Windows possono verificarsi errori di comunicazione. Per risolvere questo problema procedere come indicato di seguito in modo da ridurre il più possibile la lunghezza del buffer FIFO per la porta COM:

1)      Fare clic su "Proprietà" e aprire la scheda "Impostazioni della porta".

2)      Fare clic su "Avanzate" e impostare il buffer di ricezione e di trasmissione su "Bassa".

2.      Se le linee dei margini non vengono stampate modificare i margini con "File > Imposta pagina".

3.      Se si utilizza una stampante Kyocera FS1600 è probabile che il testo contenuto nei disegni venga stampato in modo scorretto. Per risolvere questo problema installare il driver per stampante "Kyocera FS1600+ (KPDL2)". La versione inglese del driver si trova sul DVD in <DVD-ROM>:/windows/printer/k9xpsen.exe.

Per maggiori dettagli sull'installazione dei driver per stampante consultare il manuale del sistema operativo.

4.      Per risolvere i problemi di stampa aggiornare il driver della stampante o installarne uno standard dal CD del sistema operativo.

5.      Se non si riescono comunque a risolvere i problemi di stampa salvare il file in formato PDF e stamparla con Adobe Acrobat Reader. I disegni possono essere inoltre salvati in formato BMP o JPG o modificati in altre applicazioni.

6.      Per eliminare lo sfarfallio del cursore nel modo di simulazione disattivare le impostazioni personalizzate del mouse, ad es. la traccia del puntatore.

7.      Se il nome della directory di installazione di livello superiore in un sistema operativo Windows contiene spazi, il sistema non riesce ad aprire LOGO!Soft Comfort. Assicurarsi di non utilizzare caratteri di spaziatura nel nome della directory di installazione di livello superiore.

Macintosh

Requisiti

Gli upload/download dei programmi di controllo da/in LOGO! Hardware sono stati testati con l'adattatore USB-PDA Keyspan (Keyspan USA-19) e il cavo USB per PC LOGO!.

Se LOGO!Soft Comfort non riesce a comunicare con il modulo base LOGO! tramite l'interfaccia seriale corrispondente al Keyspan USA-19 o al cavo USB per PC LOGO!, verificare se nel sistema di file è presente la cartella /var/lock. Seil file non è presente, crearlo e accertarsi di disporre dei diritti di accesso completi. Se è già stata utilizzata una versione precedente del pacchetto RXRXComm si devono cancellare dalla vecchia cartella /var/spool/uucp i file di blocco eventualmente ancora presenti. Prima di eseguire l'applicazione di setup controllare questa cartella.

Installazione

Se non si riesce ad eseguire il programma di setup Setup_Mac.command probabilmente non si dispone della necessaria autorizzazione. Questo può succedere, ad esempio, se si copia Setup_Mac.command da un computer Windows. 

Per modificare gli attributi utente del file Setup_Mac procedure nel seguente modo:

quando si avvia Setup_Mac.command compare un messaggio di errore come "Aprire il file di comando...". Confermarlo e, nella successiva finestra terminale, modificare gli attributi utente immettendo il seguente comando quando viene richiesto:

chmod a+x Setup_Mac.command

Immettere quindi la sequenza di caratteri ./Setup_Mac.command per avviare il programma di setup. Chiudere infine la finestra terminale e fare doppio clic sul file eseguibile per richiamare il programma di setup. È importante ricordare che le maiuscole e le minuscole sono rilevanti.

Stampa

MAC OS-X non supporta la stampa del formato orizzontale. Se si vuole stampare in questo formato salvare il disegno con il comando "Salva con nome" in formato PDF e stamparlo.
